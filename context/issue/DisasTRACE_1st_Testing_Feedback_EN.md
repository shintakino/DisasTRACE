## Mobile Application

### Responder

#### Account Registration

* Resubmitted registrations after rejection do not appear in CDRRMO (Super Admin).
* Registration forms are not responsive when the keyboard is open, making input fields difficult to see.

#### Home

* Top text is positioned too high.
* HELP button text is too large.
* Navigation bar is not responsive and does not move correctly.
* After selecting "Clear All" notifications, the red notification indicator still remains.
* Users can track incidents immediately even before a responder accepts them.
* Emergency incidents should be automatically dispatched, while non-emergency incidents should remain manually dispatched.

#### Responding

* Responders should have an option to accept or reject requests instead of the automatic 30-second process.
* When zooming in on the map, it automatically zooms out while responding.
* The information panel cannot be minimized or collapsed while responding.
* There is an issue with the text displayed beside the ambulance icon on the map.
* **Save As Draft:** In some cases, the device continuously vibrates because the upload process does not proceed successfully.

#### My Reports

* Response records do not appear in My Reports and are also missing from Drafts.

#### New Form

* Users cannot create a new form.

#### Profile

* **Status:** When **ON DUTY** is selected, it automatically switches back to **OFF DUTY** after a few seconds.
* Status functionality contains errors.
* The active status is not synchronized with the active indicator (dot icon) displayed on the profile.
